# F-CAN CUP Character Design Guide

## 現在の方針

キャラクターは、リアル寄りのアニメ・漫画調PNG立ち絵をメイン素材として使う。

以前のかわいいマスコット風SVGは、必要に応じて軽量表示や仮表示の予備として残している。ただし、通常のサイト表示では `assets/characters/` 配下のPNGを使う。

## 共通デザイン方針

- 函館の街を守る企業ヒーローとして見えること
- 高校生・大学生が見て「かっこいい」と感じること
- 7頭身前後のリアル寄りヒーローにすること
- 企業の仕事内容が装備、能力、色、シルエットから伝わること
- 戦闘服、現場装備、企業の仕事道具を融合させること
- 筋肉、装甲、機械部品、布、金属などの質感を出すこと
- 怖すぎず、企業イベントで使える清潔感と親しみやすさを残すこと
- カード、立ち絵、解放演出、最終決戦画面で使いやすい縦長構図にすること
- 特定作品のキャラクターや画風を直接模倣しないこと

## 各キャラクターの方向性

### スチールフレーム

翔大鋼業の鉄筋、骨組み、防御を表す重装防衛型ヒーロー。鉄骨、H鋼、建築フレームのような装甲と巨大な鋼鉄の盾を持つ。色はダークグレー、鋼色、黒、黄色。`steel-frame-v2.png` は切り抜き素材ではなく、単独立ち絵として作り直した本番候補。

### シティビルダー

森川組の土木、建築、まちづくり、地盤を表す現場型ヒーロー。建設装備と戦闘装甲を組み合わせ、重機、測量、舗装道路、アンカーの要素を持つ。色は深緑、土色、オレンジ、グレー。

### どっくメカニック

函館どっくの造船、修理、海、巨大機械を表す女性の重工系ヒーロー。船体装甲、クレーンアーム、アンカー型の多機能工具を持つ。色はネイビー、鋼色、白、鉄錆色。

### マリーナ・ヒーラー

センチュリーマリーナ函館のホテル、おもてなし、回復を表す男性ヒーロー。コンシェルジュコートと上品な白・金の回復装備をまとい、温かな光で仲間の疲れを癒やす。色はミッドナイトネイビー、白、シャンパンゴールド、琥珀色。

### エナジーリンク

東興アイテックの防水、外壁改修、塗装を表す火属性ヒーロー。黒い戦闘スーツに赤、オレンジ、琥珀色の炎のような発光回路を持ち、熱工法と防水層をイメージした装備を展開する。

### ナゾゴラ

未来の函館を襲う都市侵食型怪獣。企業の役割や街を支える仕事が忘れられたことで生まれた存在。黒緑、暗い青灰色、焦げた茶色の外殻に、黄緑色のひび割れ発光を持つ。

## サイト内での使い方

- `app-story.js` の `characterAssets` が、JS IDとPNGファイルを対応させている
- ミッションカード、ヒーロー解放モーダル、ヒーローコレクション、最終決戦は `characterSvg()` 経由でPNGを表示する
- 物語ページ内の固定カードは `index.html` に直接PNGを読み込んでいる
- サイズや配置は `styles.css` の `character-art`、`mission-character`、`scout-card .character-art`、`final-kaiju` で調整する

## 今後の注意

- 画像を差し替える場合は、同じファイル名で上書きするより、必要に応じて `-v2` などを付けて比較できるようにする
- 新しい画像を追加したら `app-story.js` の `characterAssets` とこのドキュメントを更新する
- 立ち絵は背景透過PNGにする
- 白衣、銀色装甲、水しぶきなど、白いパーツが多いキャラクターは背景除去で消えやすいので確認する
