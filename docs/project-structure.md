# F-CAN CUP / Hero Match Quest ファイル構成

このWebサイトは、静的なHTML/CSS/JavaScriptで動く謎解きサイトです。
サーバーやビルドツールは使っていません。基本的には `index.html` をブラウザで開けば動きます。

## まず見るファイル

```txt
C:\Fkyanp
├─ index.html
├─ styles.css
├─ app-story.js
└─ assets
   └─ hero-quest-map.jpg
```

## 各ファイルの役割

### `index.html`

サイトの土台です。

主に以下を担当しています。

- ページ全体のHTML構造
- ヘッダーとナビゲーション
- トップページ、受付、物語、ミッション、企業一覧、最終決戦のテンプレート
- トップページで使うメイン画像 `assets/hero-quest-map.jpg` の読み込み
- キャラクターSVGの埋め込み
- `app-story.js` の読み込み

画面に出るセクション構成を大きく変えたい時は、このファイルを見ます。

### `styles.css`

サイトの見た目をまとめているファイルです。

主に以下を担当しています。

- クリーム色と薄緑のグリッド背景
- 太い深緑の枠線
- 黄色ボタン
- カードUI
- スマホ/PCのレスポンシブ対応
- ヒーローカード、受付フォーム、ミッション画面、最終決戦画面のレイアウト
- 解放演出や最終決戦のCSSアニメーション

文字がかぶる、カードが細い、スマホで崩れるなど、見た目の問題は基本的にこのファイルを直します。

### `app-story.js`

サイトの動きとデータをまとめているメインJavaScriptです。

主に以下を担当しています。

- 6社の企業ヒーローデータ
- ミッション名、謎、ヒント、答え、企業紹介文
- 受付情報の保存
- 解放済みヒーローの保存
- 気になる企業の保存
- ヒント閲覧状況の保存
- ルーティング
- ミッション一覧の表示
- 謎解き回答判定
- ヒーロー解放モーダル
- 最終決戦の演出
- localStorageによる進行状況管理

企業情報、謎の答え、ヒント文、ヒーロー名などを変えたい時は、このファイルの `missions` 配列を見ます。

### `assets/hero-quest-map.jpg`

トップページのメインビジュアルです。

現在、以下で使っています。

- トップページの大きなメイン画像
- OGP画像指定

この画像はサイト表示に必要なので削除しないでください。

## 補助フォルダ

### `docs`

資料置き場です。

```txt
docs
├─ character-design-guide.md
├─ web-site-structure.md
└─ project-structure.md
```

各ファイルの役割は以下です。

- `character-design-guide.md`: キャラクターデザイン方針のメモ
- `web-site-structure.md`: 初期のサイト構成メモ
- `project-structure.md`: このファイル。現在のファイル構成説明

サイト表示には直接使っていません。

### `tmp`

一時作業用フォルダです。

生成途中のファイルや確認用ファイルを置く場所です。
本番サイトに必要なファイルは置かない想定です。

### `.git`

Git管理用フォルダです。

手動で編集・削除しないでください。

### `.agents`

Codexなどの作業補助用フォルダです。

通常は触らなくて大丈夫です。

## 変更したい内容ごとの見る場所

```txt
文章や画面構成を変えたい       -> index.html
色・余白・カード・スマホ崩れ   -> styles.css
企業/謎/答え/ヒントを変えたい  -> app-story.js
トップ画像を差し替えたい       -> assets/hero-quest-map.jpg
仕様メモを確認したい           -> docs
```

## 削除しないほうがよいもの

```txt
index.html
styles.css
app-story.js
assets/hero-quest-map.jpg
```

この4つは現在のサイト表示に必要です。

## 現在のサイトの起動方法

```txt
C:\Fkyanp\index.html
```

このファイルをブラウザで開くとサイトが表示されます。
